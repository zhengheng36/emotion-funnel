## 📖 项目介绍

很多时候，我们只感觉到“难受”、“心累”或“不舒服”，却无法准确描述内心发生了什么。

**情绪精准定位漏斗** 是一个基于认知心理学的轻量级 Web 工具。它摒弃了枯燥的心理测试，采用**“逐级过滤”**的逻辑，通过简单的 3 步选择，帮助你从模糊的感觉中抽丝剥茧，找到最核心的情绪词汇，并提供相应的**心理机制解析**与**行动解药**。

## ✨ 核心功能

*   **🌪️ 漏斗式定位逻辑**：
    *   **第一层**：判断效价（舒服/难受）与能量状态（高/低/排斥）。
    *   **第二层**：锁定核心情绪类别（如愤怒、恐惧、悲伤、厌恶等）。
    *   **第三层**：区分细微差别（例如：是“嫉妒”还是“暴怒”？是“焦虑”还是“惊恐”？）。
*   **💊 情绪解药**：不仅告诉你“这是什么”，还基于心理学原理告诉你“为什么”以及“怎么做”（Action）。
*   **🎨 可视化情绪轮盘**：内置高清情绪轮盘图，支持全屏放大对照查看。
*   **📱 极简交互**：支持“后悔药”功能（返回上一步），移动端/桌面端完美适配。

## 🛠️ 技术栈

本项目是一个纯静态网页，无需后台，极速加载：
*   **HTML5**：语义化结构
*   **CSS3**：响应式布局，优雅的动效
*   **JavaScript (ES6)**：模块化逻辑 (`logic.js` 处理交互, `data.js` 管理内容库)
